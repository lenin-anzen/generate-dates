# generate-dates
With MomentJS generate an array of dates and put them in a JSON File

### Installation

Install the dependencies:
```bash
npm i
```

Simple run it with:
```bash
npm start
```

### Libraries used

 * [Node.js - jsonfile](https://github.com/jprichardson/node-jsonfile)
 * [Moment.js](https://momentjs.com)
